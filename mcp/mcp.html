MCP Console

Invoke any tool directly, or connect an external MCP client using the extension ID below.

Extension ID

Web-based MCP clients call chrome.runtime.sendMessage(EXTENSION_ID, { type: "mcp", payload: <json-rpc> }, callback) to invoke tools. Copy this ID into your client config.

Send an arbitrary MCP / JSON-RPC 2.0 request.

      
      

        
      

      

    


    

      
Sample client snippet

      
const EXTENSION_ID = "…";
chrome.runtime.sendMessage(
  EXTENSION_ID,
  { type: "mcp", payload: { jsonrpc: "2.0", id: 1, method: "tools/list" } },
  (resp) => console.log(resp);
);
