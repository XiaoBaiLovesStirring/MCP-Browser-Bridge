MCP Console

Invoke tools and send raw JSON-RPC to the MCP server.

How to connect

Call chrome.runtime.sendMessage from any web page with your extension id. The extension id is shown below.

chrome.runtime.sendMessage("YOUR_ID", {
  type: "mcp",
  payload: { jsonrpc: "2.0", method: "tools/list", id: 1 }
}, (resp) => console.log(resp));
