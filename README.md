# MCP-Browser-Bridge

A bridge that connects Model Context Protocol (MCP) with the browser, enabling AI agents to interact with web content seamlessly.

## Overview

MCP-Browser-Bridge provides a standardized way for MCP-compatible AI agents to:

- Read and navigate web pages
- Execute browser actions (click, type, scroll)
- Extract structured data from web content
- Manage browser sessions and tabs

## Getting Started

### Prerequisites

- Node.js >= 18
- An MCP-compatible client

### Installation

```bash
git clone https://github.com/your-username/MCP-Browser-Bridge.git
cd MCP-Browser-Bridge
npm install
```

### Usage

```bash
npm start
```
